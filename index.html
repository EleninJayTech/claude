Claude Code는 컴퓨터에서 대화하듯 개발을 시킬 수 있는 AI 도구입니다(Anthropic 제작). 이곳의 문서들은 그 Claude Code를 내 컴퓨터에 맞게 세팅해 주는 "말로 실행되는 설명서"입니다 — 가구 설명서를 읽고 조립까지 해 주는 로봇처럼, 문서를 폴더에 넣고 "적용해줘"라고 말하면 Claude가 알아서 준비를 끝냅니다. 설정 파일을 몰라도 됩니다.

GitHub 저장소 00 문서 열기

이 그림 자체가 03 문서의 svg-design 스킬 레시피(Gooey 필터 feColorMatrix 19 -8 + Squiggle 손그림 떨림)로 그린 것입니다. 설치하면 Claude가 이런 것을 만들어 줍니다.

사용법은 세 줄입니다

복사한다 → 말한다 → 고른다

# 1. 00_통합-설치.md 파일을 내 작업 폴더에 복사해 넣고
# 2. Claude Code를 켜서 이렇게 말합니다:
"00_통합-설치.md 를 학습해서 이 프로젝트에 적용해줘"
# 3. "무엇을 설치할까요?" 목록이 나오면 원하는 것만 체크 → 끝
  • 여기서 "프로젝트"란 지금 작업 중인 폴더 하나를 말합니다. 컴퓨터 전체에 세팅하려면 "이 PC에 적용해줘"라고 말하면 됩니다.
  • 파일은 00 하나만 복사해도 됩니다 — 나머지 문서는 필요할 때 Claude가 인터넷(이 저장소)에서 가져옵니다.
  • 이미 설치된 것이 있으면 Claude가 그대로 둘지 / 고칠지 / 새로 깔지를 먼저 물어봅니다. 말없이 덮어쓰지 않습니다.

하루 루틴은 명령 두 개

# 시작할 때 — 어제 상태 불러오기
/resume ← "지난번엔 여기까지 했고, 다음은 이거죠?" 하고 AI가 먼저 보고
# 일할 때 — 확인 방법까지 같이 요청
"로그인 화면 고쳐줘. 고친 다음 제대로 되는지 확인까지 해줘"
# 끝낼 때 — 오늘 일을 일지에 기록
/wrap ← 오늘 한 일·결정이 자동으로 기록됨
  • 기록 덕분에 내일 다시 켜도 이어서 할 수 있습니다. 기록 후 저장(커밋)까지 해야 다음 날 찾습니다 — Claude가 방법을 안내합니다.
  • AI가 자꾸 헛돌면: 다른 일로 넘어갈 때 /clear(대화 초기화), 같은 문제를 두 번 고쳐줘도 안 되면 역시 /clear 후 새로 요청하는 게 빠릅니다.

자주 묻는 것

처음 보면 궁금한 것들

Claude Code가 뭔가요?

Anthropic이 만든 AI 개발 도구입니다. 채팅하듯 "이런 프로그램 만들어줘"라고 말하면 AI가 내 컴퓨터에서 직접 파일을 만들고 고칩니다. 이 저장소의 문서들은 그 도구를 더 안전하고 똑똑하게 쓰도록 세팅해 주는 것입니다.

개발을 몰라도 쓸 수 있나요?

네. 문서를 폴더에 넣고 한 문장 말하는 게 전부이고, 나머지는 Claude가 물어보면서 진행합니다. 모르는 용어가 나오면 Claude에게 "쉽게 설명해줘"라고 하면 됩니다.

문서가 왜 전부 md(텍스트 파일)인가요?

사람이 아니라 Claude가 읽고 실행하는 용도라서입니다. 사람용 안내는 이 페이지 하나로 유지합니다.

내용을 믿어도 되나요?

문서의 모든 기술 내용은 Anthropic 공식 설명서를 그날 직접 확인한 날짜와 함께 기록되고, 6개월마다 다시 점검합니다.

비밀번호 같은 건 안전한가요?

01을 설치하면 비밀번호·인증서·열쇠 파일을 AI가 읽지도, 인터넷에 올리지도 못하게 잠급니다. 그래도 근본 원칙은 "진짜 비밀은 작업 폴더에 두지 않기"입니다.

이 저장소의 기본 원칙1순위는 최적의 결과물입니다. 토큰 절약은 품질을 해치지 않는 범위의 2순위입니다.
