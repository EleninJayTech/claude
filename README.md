# Claude Code 구성 문서 저장소

> **Claude Code**는 채팅하듯 말하면 AI가 내 컴퓨터에서 직접 개발을 해 주는 도구입니다(Anthropic 제작).
> 이 저장소의 문서들은 그 Claude Code를 **내 컴퓨터·내 작업 폴더에 맞게 자동으로 세팅해 주는 "말로 실행되는 설명서"** 입니다.
> 가구 설명서를 읽고 조립까지 해 주는 로봇처럼 — 문서를 넣고 한 문장 말하면 준비가 끝납니다.

📖 **처음이라면**: [사용법 페이지(GitHub Pages)](https://eleninjaytech.github.io/claude/)를 먼저 보세요. 개발을 몰라도 이해할 수 있게 썼습니다.

## 사용법 (세 줄)

```text
1. 00_통합-설치.md 파일을 내 작업 폴더에 복사해 넣는다
2. Claude Code를 켜고 말한다:
   "00_통합-설치.md 를 학습해서 이 프로젝트에 적용해줘"
   (컴퓨터 전체 세팅이면: "... 이 PC에 적용해줘")
3. "무엇을 설치할까요?" 목록이 나오면 원하는 것만 체크한다 — 끝
```

- 파일은 **00 하나만** 복사해도 됩니다. 나머지는 Claude가 이 저장소에서 알아서 가져옵니다.
- 이미 설치된 것이 있으면 **그대로 둘지 / 고칠지 / 새로 깔지**를 먼저 물어봅니다. 말없이 덮어쓰지 않습니다.
- 모르는 용어가 나오면 Claude에게 "쉽게 설명해줘"라고 하면 됩니다.

## 이 저장소의 원칙

- **결과물의 품질이 1순위.** AI 사용료(토큰) 절약은 품질을 해치지 않는 범위의 2순위입니다.
- **기억이 아니라 확인.** 문서의 모든 기술 내용은 Anthropic 공식 설명서를 그날 직접 확인한 날짜와 함께 기록하고, 6개월마다 다시 점검합니다.
- **어디서나 똑같이.** 특정 프로젝트나 특정 컴퓨터에서만 되는 내용은 넣지 않습니다.
