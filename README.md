# Claude Code 구성 문서 저장소

> Claude Code를 **바로 잘 쓰게 만들어 주는 드롭인(Drop-in) 설치 문서** 모음.
> 문서를 프로젝트에 넣고 **"학습해서 적용해줘"** 라고 말하면, Claude가 문서를 읽고 스스로 환경을 구성합니다.

## 빠른 시작

```text
1. 00_통합-설치.md 파일을 내 프로젝트 루트(또는 docs/)에 복사
2. Claude Code 실행 후:
   "00_통합-설치.md 를 학습해서 이 프로젝트에 적용해줘"
   (PC 셋업이면: "... 이 PC에 적용해줘")
3. 선택 목록에서 원하는 것만 체크 → Claude가 설치·검증 후 보고
```

00 하나만 복사해도 됩니다 — 나머지 문서가 로컬에 없으면 이 저장소에서 자동으로 가져옵니다.
기존 구성이 감지되면 **유지 / 재구성 / 교체**를 반드시 물어보므로 덮어쓰기 걱정은 없습니다.

## 원칙

- **1순위 = 최적의 결과물.** 토큰 절약은 품질을 해치지 않는 범위의 2순위입니다.
- **기억이 아니라 검증.** 모든 기술 서술은 공식 문서(code.claude.com/docs)를 당일 조회해 확인한 날짜와 함께 기록하고, 6개월 주기로 재검증합니다(각 문서 §L).
- **범용.** 특정 프로젝트·개인 환경에 종속된 내용을 두지 않습니다. 어떤 PC, 어떤 저장소에서든 같은 결과가 나오는 것이 목표입니다.
